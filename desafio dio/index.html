Listas

Voltar

Para fazer uma lista ordenada são usadas as tags <ol> que é uma tag pai. Já dentro dela usamos <li>.

Exemplo

  1. lista
  2. lista
  3. lista
  4. lista

E para uma lista desordenada são usadas a tag <ul>

Exemplo

  • lista
  • lista
  • lista
  • lista
